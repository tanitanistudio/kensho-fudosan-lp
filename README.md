# 株式会社健匠不動産 LP

小平市・近隣7市の不動産購入層をターゲットにした、LINE登録誘導型のランディングページです。

**公開URL**: https://tanitanistudio.github.io/kensho-fudosan-lp/

## 構成

- `index.html` … ページ本体（マークアップ＋JavaScript）
- `style.css` … スタイル一式
- `assets/` … ロゴ・代表写真・スタッフ写真・SNSアイコンなどの画像
- `index_c.html` … デザイン案C（参考用の別バージョン）
- `index_backup.html` … 旧バージョンのバックアップ

## 物件情報の更新方法（重要）

トップページ「物件紹介」に表示される物件一覧は、以下のGoogleスプレッドシートの内容を自動で取得して表示しています。

**スプレッドシート**: https://docs.google.com/spreadsheets/d/16Ks5On9f_yVGqDBXQCuGswsWH0Qs0EBKgMoolocTbqE/edit?gid=0#gid=0

- このシートに行を追加・編集するだけで、サイト側は**コードを一切触らずに**物件情報を更新できます。
- サイトはページ読み込みのたびにこのシートを取得するため、シートを保存すればすぐにサイトへ反映されます（反映されない場合はブラウザのキャッシュを疑ってスーパーリロードしてください）。
- 対応している列は以下の通りです（1行目の見出し行はそのままにしてください）。

  | 列名 | 内容 | 備考 |
  |---|---|---|
  | No. | 物件番号 | 空欄不可・重複しない番号を入力 |
  | 掲載/編集中 | `掲載` または `編集中` | **`掲載`と入力した行だけサイトに表示されます。`編集中`や空欄の行はサイトに表示されません**（情報を準備中の物件を隠しておく用途） |
  | タイトル | 物件名 | **必須。空欄の行は表示されません** |
  | 住所 | 所在地 | |
  | 画像URL | 物件写真の画像URL | 空欄の場合は「間取り図」のプレースホルダー画像が表示されます |
  | 公開/未公開 | `公開` または `未公開` | `未公開`はLINE登録者のみ閲覧できる会員限定物件になります |
  | 間取り | 例：4LDK | |
  | 土地面積 | 数値のみ（㎡） | 例：110.21 |
  | 金額 | 円単位の金額（例：¥49,800,000） | サイト上は万円表示に自動変換されます |
  | 借入金・年利・返済年数・毎月の支払い目安 | 参考値 | サイトには使用していません（サイト上の「毎月の支払い目安」は、ご年齢・頭金の入力に応じてその場で自動計算されます） |

- シートの1つ目のタブ（gid=0）のみを参照しています。別タブに追加しても反映されません。
- 現在は編集・共有のURLと同じシートを直接参照しているため、シートの共有設定は「リンクを知っている全員が閲覧可」のままにしておいてください（非公開にすると物件情報が表示されなくなります）。

## ローカルでの確認方法

```bash
cd kensho-fudosan-site
python3 -m http.server 8811
# http://localhost:8811 をブラウザで開く
```

## LINE友だち追加URLなどの設定

`index.html` 内の以下の定数を、実際のURLに差し替えてください。

```js
const LINE_URL = "#";       // LINE友だち追加URL
const MYPAGE_URL = "#";     // Facilo マイページURL
const PRESCREEN_URL = "#";  // いえらぶ ダンドリ 事前審査URL
```
